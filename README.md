## Featured Projects

- **[where-is-my-data](https://github.com/Tyler-Irving/where-is-my-data)** — Interactive map of 234+ cloud and colo datacenters. AWS, GCP, Azure, Equinix, and 20+ providers with latency estimation and cost comparison. Built with Next.js 16, React 19, Mapbox GL, Tailwind v4.
- **[re-automation](https://github.com/Tyler-Irving/re-automation)** — AI content platform for real estate agents. Generates listings, Instagram posts, email campaigns, and market updates tuned to each agent's voice. Claude API + Supabase + Stripe.
- **[Kalshi Weather Trading Daemon](https://github.com/Tyler-Irving/kalshi-weather-arbitrage-daemon)** — Python daemon that scans weather data across 5 providers for edges in prediction markets and places trades automatically.
- **[Kalshi Weather Dashboard](https://github.com/Tyler-Irving/kalshi-weather-arbitrage-dashboard)** — React + Django dashboard for monitoring live weather arbitrage positions.
- **[Crypto Daemon](https://github.com/Tyler-Irving/crypto-daemon)** — RSI-based algorithmic trading bot that runs autonomously on Coinbase.
- **[File-to-API](https://github.com/Tyler-Irving/file-to-api)** — Upload a CSV or Excel file, get an instant REST API back.
